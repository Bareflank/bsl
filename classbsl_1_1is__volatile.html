Detailed Description

template<typename T>
class bsl::is_volatile< T >

If the provided type is qualified as volatile, provides the member constant value equal to true. Otherwise the member constant value is false.

Note
"volatile" is not supported by the BSL as it is not compliant with AUTOSAR. We only provide this function so that it can be used to detect if volatile is used in order to throw a compilation error. Do not use "volatile" with the BSL.
